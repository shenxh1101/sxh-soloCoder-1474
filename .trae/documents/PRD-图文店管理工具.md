## 1. 产品概述

专为小型图文店设计的账目管理工具，解决复印打印服务记账、客户挂账管理、营收统计及耗材提醒等日常运营需求，帮助店主高效管理业务往来。

- 解决问题：手工记账混乱、客户挂账难追踪、营收统计繁琐、耗材库存易疏忽
- 目标用户：图文店店主及店员
- 产品价值：提升记账效率，减少漏账错账，清晰掌握经营状况

## 2. 核心功能

### 2.1 用户角色

本工具为单用户系统，无需角色区分，店主/店员拥有全部操作权限。

### 2.2 功能模块

1. **首页仪表板**：今日/本周/本月营收概览、快捷记账入口、耗材提醒
2. **记账管理**：新订单录入、服务项目选择、金额计算、结算方式选择
3. **客户管理**：客户信息维护、挂账记录查询、还款登记、账单打印
4. **服务价格**：复印、打印、彩印、扫描、装订等服务单价设置
5. **常用模板**：高频客户的常用服务模板保存与快速调用
6. **耗材提醒**：打印机粉量、纸张库存手动登记与提醒

## 3. 核心流程

### 3.1 日常记账流程

```mermaid
flowchart LR
    A["开始记账"] --> B["选择客户类型<br/>（散客/挂账客户）"]

    F --> G["选择结算方式<br/>（现金/挂账）"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色 (#1e40af) - 代表专业、可信赖
- **辅助色**：琥珀色 (#f59e0b) - 用于提醒和强调操作
- **成功色**：翠绿色 (#10b981) - 现金收入、已还款
- **警告色**：橙红色 (#ef4444) - 挂账、低库存提醒
- **按钮样式**：圆角矩形 (6px)，悬停时有轻微上浮效果
- **字体**：系统无衬线字体，标题加粗，正文清晰易读
- **布局风格**：卡片式布局，左侧导航栏 + 右侧内容区
- **图标风格**：简洁线性图标，配合emoji增强识别度

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 首页仪表板 | 营收概览 | 三张统计卡片并排，大数字展示金额，背景色区分现金/挂账/总计 |
| 首页仪表板 | 快捷操作 | 四个大按钮网格排列，图标+文字 |
| 首页仪表板 | 耗材提醒 | 进度条显示库存，低于阈值时红色闪烁警告 |
| 记账管理 | 订单录入 | 表单布局，服务项目用卡片选择，数量输入带加减按钮 |
| 客户管理 | 客户列表 | 表格展示，欠款金额用红色标注，支持搜索筛选 |
| 客户管理 | 挂账明细 | 时间线样式展示每笔交易，汇总区显示总欠款 |
| 服务价格 | 价格配置 | 每行一个服务项目，单价可编辑，实时保存 |

### 4.3 响应式设计

- 桌面端优先设计，主内容区最小宽度1024px
- 平板端：导航栏折叠为汉堡菜单，统计卡片换行显示
- 移动端：单列布局，表格改为卡片列表展示
- 所有交互元素确保最小44x44px触控区域

### 4.4 打印优化

- 对账单打印时隐藏导航栏和操作按钮
- 使用打印专用样式，确保黑白打印清晰可读
- 页面设置A4尺寸，预留装订边距
